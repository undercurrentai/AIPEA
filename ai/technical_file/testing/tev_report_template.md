# TEVV Report Template

## Scope
- Model/version; environment; date

## Datasets & Splits
- Train/val/test; bias/fairness evaluation datasets

## Metrics
- Accuracy/quality (with confidence intervals)
- Latency & throughput (p95/p99)
- Robustness: noise injection, adversarial prompts, OOD

## Security Tests
- Prompt injection, data exfiltration attempts, SSRF/egress tests

## Bias & Fairness
- Methodology and results by protected/impactful slices (jurisdiction-appropriate)

## Regression Comparison
- Δ vs previous baseline; gate decisions

## Conclusions & Actions
- Risks, mitigations, follow-ups
